# Number Guessing Game (C++)

## Description
A console-based number guessing game developed in C++ where the player
has limited attempts to guess a randomly generated number. The program
provides feedback and calculates score based on remaining attempts.

## Features
- Random number generation
- Limited attempts
- Score tracking
- Input validation

## Technologies Used
- C++

## How to Run
g++ number_guessing_game.cpp -o game
./game
